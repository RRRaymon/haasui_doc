## 其它基本样式

## Opacity

`opacity {number}`：取值范围为 [0, 1] 区间。默认值是 1，即完全不透明；0 是完全透明；0.5 是 50% 的透明度。



## background-color

 `RGBA（ `rgba(255, 0, 0, 0.5)` ）;` 

 `十六进制（ `#ff0000` ）;` 

 `精简写法的十六进制（ `#f00` ）;`
